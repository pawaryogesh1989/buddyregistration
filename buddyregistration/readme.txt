== Description ==

This plugin provides buddypress registration form as a widget. It also disable the cover image of group and makes the profile pic manadatory for all users (except super admin).

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/buddyregistration` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
